# SkillSprint AI — Frontend

React app (Create React App + Craco) with Tailwind CSS and shadcn/ui.

## Available scripts

- `npm start` — run the dev server at http://localhost:3000
- `npm run build` — production build to `build/`
- `npm test` — run tests

Set `REACT_APP_BACKEND_URL` in `.env` (see `.env.example`) to point at your backend.
